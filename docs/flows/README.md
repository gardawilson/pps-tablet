# Flows

Folder ini untuk seluruh flow diagram sistem.

## Konvensi

Untuk setiap flow, buat subfolder sendiri:

- `docs/flows/<nama-flow>/README.md` -> penjelasan narasi
- `docs/flows/<nama-flow>/<nama-flow>.mmd` -> source mermaid
- `docs/flows/<nama-flow>/<nama-flow>.png` -> hasil render gambar

Contoh:

- `docs/flows/print-lock/`
